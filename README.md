# OrdinaryDay

수면시간 입력하지 않으면 메시지 창 띄우기

수면시간 입력하면 '나는 ?시간 잠을 잡니다'의 ?를 채움


코딩 독서 영어 운동 시간을 합쳐서 

'나는 꿈을 위해 ?시간 투자합니다' 의 ?를 채울 것

마지막에 내가 하고 있는 활동의 그림들 띄울 것 


스피너를 활용해 이상형을 입력하면 

'나의 이상형은 Appearance(외모)입니다'를 작성
